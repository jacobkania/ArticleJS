This could be an important headline.

ArticleJS allows you to implement full-blown HTML into your pages. Use any tags that you're comfortable with. Since it's just injecting HTML into your page from a file, all of your websites styles will be applied to the article automatically.

I am working to make a minimalist front-end platform to load full-HTML news articles into a webpage, paired with tools to write and manage articles.

This is still an early work-in-progress, and all pull requests will be considered! Feel free to talk to me about any features that you're looking at adding. I've been developing software for several years, but this is my first big project with javascript. You can find my github profile here if you'd like to chat!
